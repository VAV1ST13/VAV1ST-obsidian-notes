[[00 Obsidian]]

*личный опыт:*
```
1. Информация никогда "не забывается" и легко находится.
т. к. всё хранится в "одном файле", конспекты не рассыпаются по всему пространству жесткого диска (а их стараюсь сортировать). 

2. Со временем (примерно после полугода) где расположена информация "забывается"/"теряется из виду" и "теряется".
Что сильно затрудняет их поиск.
(это про обычное хранение конспектов, не Obsidian)

3. Иногда становится необходимым применить знания из конспектов в какой либо деятельности, и можно провести линк от "конспекта" к "выполняемой задаче" 
(если она расписана в Obsidian)

4. Возможность одним действием "копировать-вставить" делится своими наработками с друзьями/знакомыми по каким либо вопросам - бесценно.
(а не заниматься писаниной/расписыванием текста заново - всё равно что в десятый раз "изобретать велосипед" если делишься информацией неоднократно на протяжении неск. месяцев/лет)
```

```
преимущества:
1. загрузка в память "забытых" знаний после перечитывания заметок
2. переход количество в качество (возможность видеть связи как разные разделы связаны между собой)
3. персонализация онтологии (база знаний становятся зеркалом мозга)
4. ссылки (на др. заметки) только по делу
5. ссылки на пруфы
6. для таблиц и картинок лучше использовать отдельный спец. софт (прим. Excel)
```
